Dans le cas où vous envisageriez de transiter, pour une raison ou une autre, par différents pays européens dans le cadre de vos études, cette section vous est destinée. Plusieurs cas seront évoqués avec la description des démarches qui sont liées.


## 9.1 Cas du double diplôme universitaire

Dans le cas d’un double diplôme (1) faisant intervenir une université française et une autre au sein de l’union européenne vous devrez au préalable faire une demande de visa long séjour d’un an avant de quitter votre pays de départ. Par la suite, durant le parcours universitaire vous serez éventuellement amené à faire un stage dans l’établissement avec lequel l’université d’origine a passé un accord de partenariat pour le double diplôme. Dans certains cas le stage peut déboucher sur un emploi.
